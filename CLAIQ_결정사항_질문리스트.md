## 우선 답변 필요 TOP 5

개발 첫 날 바로 시작하려면 이 5개가 가장 급합니다:

1. **Supabase 프로젝트 URL + anon key** (DB 스키마 생성)
2. **OpenAI API 키** (AI 파이프라인)
3. **GitHub 저장소 이름** (프로젝트 셋업)
4. **지원 과목 범위** (DB 스키마 설계 영향)
5. **메인 색상 테마** (프론트엔드 Tailwind config)
