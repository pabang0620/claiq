## 발견 및 수정된 버그 전체 목록

### 1차 실서버 테스트에서 발견 (6건)

| 버그 | 증상 | 원인 | 수정 내용 |
|------|------|------|---------|
| BUG-2 | 교강사 승인/반려 → 400 오류 | 프론트 `action` 전송, 백엔드 `status` 기대 | `question.api.js` 수정, 백엔드 enum 양쪽 수용 |
| BUG-3 | 수강생 문제 제출 → 400 오류 | `academy_id` 필수였으나 미전송 + 타입 불일치 | `submitSchema` optional 처리, `::integer` 캐스팅 |
| BUG-4 | AI Q&A 세션 생성 → 404/400 | `POST /qa/sessions` 라우트 없음 | 라우트 추가, `resolveAcademyAndTeacher()` 헬퍼 구현 |
| BUG-5 | 미니 모의고사 → 무한 루프 토스트 | 생성 실패 시 즉시 재호출 루프 | `hasError` 플래그로 루프 차단, 백엔드 auto-resolve |
| BUG-6 | 약점 분석 → 흰 화면 크래시 | `typeName.length` undefined 접근 | 옵셔널 체이닝 · nullish fallback 적용 |
| BUG-7 | 이탈 대시보드 vs 상세 불일치 | `academy_id` 미전달로 빈 결과 반환 | `resolveAcademyId()` 적용 |

### 2차 로컬 테스트에서 발견 (4건)

| 버그 | 증상 | 원인 | 수정 내용 |
|------|------|------|---------|
| BUG-8 | AI Q&A 스트리밍 중단 | `signal`을 body에 잘못 전달 | `ragQA.js` — signal을 2번째 인자(options)로 이동 |
| BUG-9 | 미니 모의고사 문제 목록 미반환 | 생성 후 questions 미포함 응답 | `examService.js` — `findExamById()` 추가 조회 |
| BUG-10 | 미니 모의고사 타임아웃 | GPT 생성 65~86초, 기본 타임아웃 30초 초과 | `exam.api.js` · `examStore.js` — timeout 120초로 증가 |
| BUG-11 | 강의 업로드 multer 필드 불일치 | 프론트 `'file'` 전송, 백엔드 multer `'audio'` 기대 | `LectureUploadPage.jsx` — `'file'` → `'audio'` |
