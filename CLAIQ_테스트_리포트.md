## 발견 및 수정된 버그 목록

| 버그 | 증상 | 원인 | 수정 내용 |
|------|------|------|---------|
| BUG-2 | 교강사 승인/반려 → 400 오류 | 프론트가 `action` 전송, 백엔드는 `status` 기대 | `question.api.js` 수정, 백엔드 enum에 양쪽 형식 수용 |
| BUG-3 | 수강생 문제 제출 → 400 오류 | `academy_id` 필수였으나 미전송 + 타입 불일치 | `submitSchema` optional 처리, `::integer` 캐스팅 추가 |
| BUG-4 | AI Q&A 세션 생성 → 404/400 | `POST /qa/sessions` 라우트 없음, 파라미터 불일치 | 라우트 추가, `resolveAcademyAndTeacher()` 헬퍼 구현 |
| BUG-5 | 미니 모의고사 → 무한 루프 토스트 | 생성 실패 시 즉시 재호출 루프, `subject_id` 미전송 | `hasError` 플래그로 루프 차단, 백엔드 auto-resolve |
| BUG-6 | 약점 분석 → 흰 화면 크래시 | `typeName.length` undefined 접근 | 옵셔널 체이닝 · nullish fallback 적용 |
| BUG-7 | 이탈 대시보드 vs 상세 불일치 | `academy_id` 미전달로 빈 결과 반환 | `resolveAcademyId()` 적용, camelCase 변환 추가 |

| 항목 | 내용 | 담당 |
