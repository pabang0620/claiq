## 운영자 (operator@demo.claiq.kr)

### 대시보드
- [ ] 총 수강생 수 표시
- [ ] 이탈위험 수강생 수 표시
- [ ] 이번 달 출석률 표시
- [ ] 미발송 리포트 수 표시
- [ ] 이탈위험 목록 클릭 → 상세 페이지 이동

### 학원 설정
- [ ] 학원명 조회
- [ ] 학원 코드(STAR01) 표시
- [ ] 수능일 설정/조회
- [ ] 학원 정보 수정 후 저장

### 쿠폰 관리 *(신규 개선)*
- [ ] 쿠폰 목록 조회
- [ ] 쿠폰 생성 - 할인 유형 선택 (% / 정액)
- [ ] 쿠폰 생성 - 할인값 입력
- [ ] 쿠폰 생성 - 유효기간(일) 설정
- [ ] 생성된 쿠폰에 "30% 할인" 또는 "5,000원 할인" 형식 표시
- [ ] 만료일(YYYY.MM.DD) 표시
- [ ] 쿠폰 삭제

### 회원 관리
- [ ] 수강생 / 교강사 목록 조회 (운영자 1명 · 교강사 2명 · 수강생 5명)
- [ ] 회원 역할/상태 표시

### 이탈위험 분석
- [ ] 이탈위험 수강생 목록 조회
- [ ] at_risk / inactive 분류 표시

### 강의 통계
- [ ] 강의별 출석 수, 문제 수, 정답률 조회

### 성취 리포트
- [ ] 수강생별 리포트 생성 버튼
- [ ] 생성된 리포트 카드 표시 (출석률 · 정답률 · 포인트)
- [ ] SMS 발송 버튼 → "SMS 발송 완료" 표시 (시뮬레이션)
- [ ] 발송 완료 후 버튼 비활성화 + 시각 표시

## 교강사 (teacher1@demo.claiq.kr)

### 대시보드
- [ ] 담당 강의 수 표시
- [ ] 미검수 문제 수 표시

### 강의 업로드 *(Supabase 버킷 생성 후 활성화)*
- [ ] 강의 제목 입력
- [ ] 과목 선택 (국어/수학/영어/사회/과학)
- [ ] 오디오 파일 드롭존 업로드 (mp3/m4a/wav, 25MB 이하)
- [ ] 업로드 후 SSE 진행상황 표시: `업로드 중 → STT 변환 → 임베딩 → 문제 생성`
- [ ] 처리 완료 후 생성된 문제 수 안내 토스트

### 문제 검수 (Human-in-the-Loop)
- [ ] pending 문제 목록 조회
- [ ] 문제 상세 내용 확인 (지문 · 선택지 · 정답 · 해설 · 난이도)
- [ ] 문제 승인 → approved로 상태 변경
- [ ] 문제 반려 → rejected로 상태 변경
- [ ] 승인된 문제만 수강생에게 노출 (approved 필터 확인)

### 출결 관리
- [ ] 날짜 선택 (오늘 이후 선택 불가)
- [ ] 수강생별 출석/결석/지각/공결 처리
- [ ] 일괄 저장 (bulk) 동작
- [ ] 처리 후 출석 통계 (출석 N명 · 결석 N명 · 지각 N명) 갱신

### 강의 자료
- [ ] 강의 드롭다운 선택 (5개 강의)
- [ ] 자료 목록 조회
- [ ] PDF/이미지 파일 업로드

### Q&A 에스컬레이션
- [ ] AI가 에스컬레이션한 질문 목록 조회
- [ ] 직접 답변 작성

## 수강생 (s1@demo.claiq.kr)

### 대시보드
- [ ] 수능 D-day 카운터 표시
- [ ] 학습 스트릭 (현재/최장) 표시
- [ ] 포인트 잔액 표시
- [ ] 로드맵 진행률 표시
- [ ] 빠른 링크 (오늘의 문제 · Q&A · 모의고사) 동작

### 오늘의 문제
- [ ] approved 문제만 로드 (N/7 형식)
- [ ] 5지선다 선택 후 제출
- [ ] 단답형 입력 후 제출
- [ ] 정답/오답 표시 + 해설
- [ ] 다음 문제 이동
- [ ] 전체 완료 후 결과 페이지 (맞은 수 · 유형별 분석)
- [ ] 정답 시 포인트 적립 안내

### AI Q&A
- [ ] 질문 입력 후 전송
- [ ] 스트리밍 답변 (타이핑 효과) 표시
- [ ] 이전 대화 히스토리 유지
- [ ] 에스컬레이션 발생 시 안내 메시지

### 약점 분석
- [ ] 수능 유형별 정답률 차트
- [ ] 취약 유형 Top 3~5 목록

### 학습 로드맵
- [ ] D-day 역산 주간 학습 계획 조회
- [ ] 항목 완료 체크

### 미니 모의고사
- [ ] 모의고사 시작 (GPT 생성, 최대 120초 대기)
- [ ] 15문항 표시
- [ ] 문제 풀기 (타이머 동작)
- [ ] 제출 후 결과 페이지 (점수 · 유형별 분석)

### 강의 자료
- [ ] 교강사가 업로드한 자료 목록 조회
- [ ] 자료 열람/다운로드

### 포인트 내역
- [ ] 포인트 잔액 조회
- [ ] 적립/사용 내역 (이유 · 포인트 · 날짜)

### 뱃지 *(신규 개선)*
- [ ] 획득한 뱃지 강조 표시 (컬러 · "획득" 라벨)
- [ ] 미획득 뱃지 흐리게 표시
- [ ] 획득 뱃지 클릭 시 획득일 표시
- [ ] 스트릭 카드 (현재 N일 · 최장 N일)
- [ ] 뱃지 진행 안내 메시지 (N개 남았어요)
- [ ] 문제 풀기 완료 시 자동 뱃지 체크 (QUIZ_FIRST 등)
- [ ] 모의고사 완료 시 자동 뱃지 체크
