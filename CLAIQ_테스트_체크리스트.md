## ① 공통 / 인증

> 로그인 화면에서 시작

- [ ] **회원가입** `/signup` 진입 → 운영자 역할 선택 → 이름·이메일·비밀번호 입력 → 가입 완료
- [ ] **회원가입** → 교강사 역할 선택 → 가입 → 학원 코드 입력 화면(`/join-academy`) 진입 확인
- [ ] **회원가입** → 수강생 역할 선택 → 가입 → 학원 코드 입력 화면 진입 확인
- [ ] **데모 로그인** 로그인 화면 하단 "운영자 · 정민석" 버튼 클릭 → 자동 로그인 후 `/operator` 이동
- [ ] **데모 로그인** "교강사 · 이준혁" 버튼 → `/teacher` 이동
- [ ] **데모 로그인** "수강생 · 김민준" 버튼 → `/student` 이동
- [ ] **로그아웃** 헤더 아바타 클릭 → 로그아웃 → 로그인 화면 이동
- [ ] **권한 차단** 수강생 로그인 상태에서 브라우저 주소창에 `/operator` 입력 → "접근 권한이 없어요" 화면
- [ ] **이용약관** 로그인 화면 하단 링크 `/terms` → 페이지 렌더링
- [ ] **개인정보처리방침** `/privacy` → 페이지 렌더링

## ② 운영자 (operator@demo.claiq.kr)

> 로그인 후 `/operator` 시작

### 대시보드

- [ ] **통계 카드 4개** 표시 확인
  - 전체 수강생 수
  - 이탈위험 수강생 수
  - 이번 달 출석률(%)
  - 미발송 리포트 수
- [ ] **이탈위험 카드** 클릭 → `/operator/churn` 이동 확인 (사이드바 메뉴로도 진입 가능)

### 이탈위험 분석 `/operator/churn`

- [ ] 수강생 목록 표시 (at_risk / inactive 상태)
- [ ] 각 카드에 **비활성 일수**, **출석률**, **위험 점수** 표시
- [ ] 수강생 카드 클릭 → 상세 정보 확인

### 강의 통계 `/operator/stats`

- [ ] 강의 목록 (제목 · 교강사 · 출석 수 · 문제 수 · 정답률) 표시
- [ ] 과목 필터 클릭 → 목록 필터링 확인
- [ ] 기간 필터 (7일/30일/90일) 변경 → 데이터 변경 확인

### 회원 관리 `/operator/members`

- [ ] 운영자 1명 · 교강사 2명 · 수강생 5명 목록 표시
- [ ] 탭 (전체 / 교강사 / 수강생) 클릭 → 필터링
- [ ] 각 회원 카드에 이름 · 역할 · 이메일 · 가입일 표시

### 학원 설정 `/operator/settings`

- [ ] 학원명 · 학원코드(STAR01) · 수능일 표시
- [ ] 학원명 수정 후 저장 → 성공 토스트
- [ ] **쿠폰 생성**
  - [ ] 쿠폰 이름 입력
  - [ ] 할인 유형 선택 (% 할인 / 정액 할인)
  - [ ] 할인값 입력
  - [ ] 유효기간(일) 입력
  - [ ] 생성 후 목록에 "30% 할인" 또는 "5,000원 할인" 형식으로 표시
  - [ ] 만료일 (YYYY.MM.DD) 표시
- [ ] **쿠폰 삭제** 삭제 버튼 클릭 → 목록에서 제거

### 성취 리포트 `/operator/report`

- [ ] 리포트 카드 목록 표시 (수강생명 · 기간 · 출석률 · 정답률 · 포인트)
- [ ] **리포트 생성** 수강생 선택 후 "생성" 버튼 → 새 카드 추가
- [ ] **SMS 발송** 발송 버튼 클릭 → "SMS 발송 완료" 토스트 표시 (시뮬레이션)
- [ ] 발송 완료 후 버튼 비활성화 + 발송 시각 표시

## ③ 교강사 (teacher1@demo.claiq.kr)

> 로그인 후 `/teacher` 시작

### 대시보드

- [ ] **통계 카드** 표시 확인
  - 담당 강의 수
  - 미검수 문제 수
  - 이번 주 출석률
  - 에스컬레이션 수
- [ ] **빠른 링크** (강의 업로드 · 문제 검수 · 출결 관리) 클릭 → 각 페이지 이동

### 강의 목록 `/teacher/lectures`

- [ ] 5개 강의 카드 표시 (제목 · 과목 · 날짜 · 문제 수)

### 강의 업로드 `/teacher/upload`

- [ ] 강의 제목 입력 필드 표시
- [ ] 과목 드롭다운 (국어/수학/영어/사회/과학)
- [ ] 파일 드롭존 표시 (파일 드래그 또는 클릭 업로드)
- [ ] **실제 오디오 파일 업로드** (mp3/wav/m4a, 25MB 이하)
  - [ ] 업로드 버튼 클릭 후 진행 표시줄 등장
  - [ ] SSE 단계별 표시: 업로드 중 → STT 변환 → 임베딩 → 문제 생성
  - [ ] 완료 후 "N개 문제가 생성됐습니다!" 토스트
  - [ ] "새 강의 업로드" 버튼 등장

### 문제 검수 `/teacher/review`

- [ ] **pending 탭** 대기 문제 목록 표시
- [ ] **문제 카드**에 지문 · 선택지 · 난이도 · 유형 표시
- [ ] 카드 클릭 → 상세 페이지 이동
- [ ] **상세 페이지** `/teacher/review/:id`
  - [ ] 문제 전체 내용 표시 (지문 · 선택지 · 정답 · 해설)
  - [ ] **문제 수정** 내용 편집 후 저장
  - [ ] **승인** 버튼 클릭 → approved 전환 → 목록에서 사라짐
  - [ ] **반려** 버튼 클릭 → rejected 전환
- [ ] **approved 탭** 승인된 문제 목록 확인
- [ ] **rejected 탭** 반려된 문제 목록 확인

### 출결 관리 `/teacher/attendance`

- [ ] 날짜 선택 (오늘 이후 선택 불가 확인)
- [ ] 수강생 목록과 출결 상태 드롭다운 표시
- [ ] 출결 상태 변경 (출석 / 결석 / 지각 / 공결)
- [ ] **저장** 버튼 클릭 → 성공 토스트
- [ ] 상단 통계 (출석 N명 · 결석 N명 · 지각 N명) 갱신 확인

### 강의 자료 `/teacher/materials`

- [ ] 강의 드롭다운 (5개 강의) 표시
- [ ] 강의 선택 → 해당 강의 자료 목록 표시
- [ ] **파일 업로드** PDF/이미지 업로드 → 목록에 추가

### Q&A 에스컬레이션 `/teacher/escalation`

- [ ] 에스컬레이션된 질문 목록 표시
- [ ] 질문 카드에 학생 질문 · AI 시도 내용 표시
- [ ] **답변 작성** 텍스트 입력 후 제출 → 답변 완료 표시

## ④ 수강생 (s1@demo.claiq.kr)

> 로그인 후 `/student` 시작

### 대시보드

- [ ] **D-day 카운터** (수능까지 D-XXX) 표시
- [ ] **학습 스트릭** 현재/최장 연속 일수 표시
- [ ] **포인트** 잔액 표시
- [ ] **로드맵 진행률** 표시
- [ ] **빠른 링크** (오늘의 문제 · AI Q&A · 미니 모의고사) 클릭 → 각 페이지 이동

### 오늘의 문제 `/student/quiz`

- [ ] 문제 목록 로드 (N/M 형식 표시)
- [ ] **5지선다 문제** 선택지 클릭 → 선택 표시
- [ ] **단답형 문제** 입력창에 답 입력
- [ ] **제출** 버튼 클릭
  - [ ] 정답 시 "정답입니다!" + 초록 표시 + 해설
  - [ ] 오답 시 "틀렸습니다." + 빨간 표시 + 정답·해설
- [ ] **다음 문제** 버튼 → 다음 문제 이동
- [ ] 마지막 문제 제출 후 **결과 페이지** 이동
  - [ ] 맞은 수 / 전체 수 / 정답률 표시
  - [ ] 유형별 분석 표시

### AI Q&A `/student/qa`

- [ ] 질문 입력창 표시
- [ ] 질문 입력 후 Enter 또는 전송 버튼
- [ ] **스트리밍 답변** 타이핑 효과로 답변 표시
- [ ] 대화 이어서 추가 질문 → 이전 대화 컨텍스트 유지
- [ ] **새 세션** 시작 → 이전 세션 목록에 저장
- [ ] 이전 세션 클릭 → 해당 대화 내역 복원

### 약점 분석 `/student/weak`

- [ ] 수능 유형별 정답률 **차트** 표시
- [ ] 취약 유형 목록 (정답률 낮은 순) 표시
- [ ] 각 유형 항목에 정답률 · 시도 수 표시
- [ ] 색상 구분: 빨강(<50%) · 노랑(50~70%) · 초록(70%+)

### 학습 로드맵 `/student/roadmap`

- [ ] D-day 및 주간 학습 계획 타임라인 표시
- [ ] 각 주차에 학습 항목·과제 표시
- [ ] 항목 **완료 체크** 클릭 → 완료 표시로 변경
- [ ] **로드맵 재생성** 버튼 클릭 → 새 계획 생성

### 미니 모의고사 `/student/exam`

- [ ] **시작** 버튼 클릭 → GPT 문제 생성 대기 (최대 2분)
- [ ] 15문항 표시 확인
- [ ] **20분 타이머** 카운트다운 표시
- [ ] 문제 번호 버튼으로 자유 이동
- [ ] 답 선택 → 번호 버튼 색상 변경 (답변 완료 표시)
- [ ] **제출** 버튼 → 결과 페이지 이동
  - [ ] 점수 · 정답 수 / 15 표시
  - [ ] 문제별 정답·오답 표시 + 해설

### 강의 자료 `/student/materials`

- [ ] 교강사가 업로드한 자료 목록 표시
- [ ] 자료 카드에 파일명 · 강의명 · 업로드일 표시
- [ ] **열기** 버튼 → 파일 외부 링크 열림

### 포인트 내역 `/student/points`

- [ ] **잔액 · 오늘 획득 · 누적 획득** 표시
- [ ] 포인트 거래 내역 목록 (이유 · 금액 · 날짜)
- [ ] 적립(+): 초록색 / 사용(-): 빨간색 구분
- [ ] **쿠폰 교환** 버튼 클릭 → 교환 가능 리워드 목록 모달 표시
- [ ] 잔액 충분 시 "교환" 버튼 활성화 → 클릭 → 포인트 차감 + 내역 갱신

### 뱃지 & 스트릭 `/student/badges`

- [ ] **스트릭 카드** 현재 N일 · 최장 N일 표시
- [ ] **뱃지 그리드** 7개 표시
  - 획득한 뱃지: 컬러 + "획득" 라벨 + 획득일
  - 미획득 뱃지: 흑백 + 흐리게
- [ ] `QUIZ_FIRST` 뱃지 확인 (문제 풀기 완료 시 자동 부여)
- [ ] 진행 안내 메시지 "N개 뱃지가 남았어요" 표시
