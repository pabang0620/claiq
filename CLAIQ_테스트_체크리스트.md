## 공통

- [ ] 회원가입 — 운영자 역할 선택 후 가입
- [ ] 회원가입 — 교강사 역할 선택 후 가입
- [ ] 회원가입 — 수강생 역할 선택 후 가입
- [ ] 학원 코드로 학원 가입 (교강사/수강생 가입 후 코드 입력)
- [ ] 로그인 — 이메일/비밀번호 로그인
- [ ] 로그인 — 데모 계정 간편 로그인 버튼 (운영자/교강사/수강생)
- [ ] 로그아웃
- [ ] 권한 없는 페이지 접근 시 차단 (예: 수강생이 /operator 접근)
- [ ] 토큰 만료 시 자동 갱신 후 요청 재시도
- [ ] 이용약관 페이지 (/terms)
- [ ] 개인정보 처리방침 페이지 (/privacy)

## 운영자 (operator@demo.claiq.kr)

### 대시보드
- [ ] 총 수강생 수 표시
- [ ] 이탈위험 수강생 수 표시
- [ ] 이번 달 출석률 표시
- [ ] 미발송 리포트 수 표시

### 학원 설정
- [ ] 학원명 조회
- [ ] 수능일 조회
- [ ] 학원 정보 수정 후 저장

### 회원 관리
- [ ] 수강생 / 교강사 목록 조회
- [ ] 회원 상태 변경 (활성/비활성)

### 이탈위험 분석
- [ ] 이탈위험 수강생 목록 조회
- [ ] 비활성 일수 기준 정렬 확인

### 강의 통계
- [ ] 강의별 출석 수, 문제 수, 정답률 조회
- [ ] 최근 20개 강의 표시

### 성취 리포트
- [ ] 수강생별 리포트 생성 버튼 클릭
- [ ] 생성된 리포트 카드 목록 표시
- [ ] 개별 SMS 발송 버튼 동작 확인 (mock)
- [ ] 일괄 SMS 발송 버튼 동작 확인 (mock)

## 교강사 (teacher1@demo.claiq.kr)

### 대시보드
- [ ] 담당 강의 수 표시
- [ ] 미답변 질문 수 표시
- [ ] 에스컬레이션 수 표시

### 강의 업로드
- [ ] 오디오 파일 업로드 (mp3/m4a/wav, 25MB 이하)
- [ ] 업로드 후 SSE 진행상황 표시 (STT → 임베딩 → 문제생성)
- [ ] 처리 완료 후 문제 생성 확인

### 문제 검토
- [ ] 생성된 문제 목록 조회 (pending 상태)
- [ ] 문제 상세 보기
- [ ] 문제 승인
- [ ] 문제 수정 후 승인
- [ ] 문제 반려

### 강의 자료
- [ ] PDF/이미지 자료 업로드
- [ ] 업로드된 자료 목록 조회

### 출결 관리
- [ ] 수강생별 출석/결석/지각/공결 처리
- [ ] 출결 저장

### Q&A 에스컬레이션
- [ ] AI가 에스컬레이션한 질문 목록 조회
- [ ] 질문에 직접 답변 작성

## 수강생 (s1@demo.claiq.kr)

### 대시보드
- [ ] 포인트 잔액 / 총 획득 포인트 표시
- [ ] 학습 스트릭 (현재/최장) 표시
- [ ] 수능 D-day 카운터 표시
- [ ] 로드맵 진행률 표시

### 오늘의 문제
- [ ] 문제 목록 로드 (승인된 문제)
- [ ] 5지선다 객관식 선택
- [ ] 단답형 입력
- [ ] 제출 후 정답/오답 표시
- [ ] 해설 표시
- [ ] 다음 문제로 이동
- [ ] 결과 페이지 — 맞은 수 / 유형별 분석

### AI Q&A
- [ ] 질문 입력 후 전송
- [ ] 스트리밍 답변 표시
- [ ] 대화 히스토리 유지
- [ ] 에스컬레이션 발생 시 안내 메시지

### 약점 분석
- [ ] 수능 유형별 정답률 차트 조회
- [ ] 취약 유형 Top 5 표시

### 학습 로드맵
- [ ] 개인 로드맵 항목 목록 조회
- [ ] 항목 완료 체크

### 미니 모의고사
- [ ] 미니 모의고사 시작
- [ ] 문제 풀기 (타이머 동작)
- [ ] 제출 후 결과 페이지

### 강의 자료
- [ ] 교강사가 업로드한 자료 목록 조회
- [ ] 자료 다운로드/열람

### 포인트
- [ ] 포인트 잔액 조회
- [ ] 포인트 적립/사용 내역 조회

### 뱃지
- [ ] 획득한 뱃지 목록 조회
