# ADR-001: AI 모델 선택

## 상태

**승인됨** — 2025년 12월

---

## 맥락

CLAIQ의 핵심 기능인 수능 문제 자동 생성에 사용할 AI 모델을 결정해야 했습니다. 수능 문제는 교육적 정확성과 유형 적합성이 매우 중요하기 때문에, 단순히 "가장 좋은 모델"을 선택하는 것이 아니라 **품질과 비용의 균형**을 신중하게 검토할 필요가 있었습니다.

### 검토된 선택지

| 모델 | 입력 비용 (1M 토큰) | 출력 비용 (1M 토큰) | 특징 |
|------|---------------------|---------------------|------|
| gpt-4o | $2.50 | $10.00 | 최고 품질, 높은 비용 |
| gpt-4o-mini | $0.15 | $0.60 | 97% 저렴, 충분한 품질 |
| gpt-3.5-turbo | $0.50 | $1.50 | 저렴하지만 구조화 약함 |
| Claude 3 Haiku | $0.25 | $1.25 | 빠름, OpenAI 생태계와 분리 |

### 주요 고려 사항

1. **비용**: 교강사 1인이 강의 100개를 업로드했을 때 감당 가능한 비용이어야 한다
2. **품질**: 생성된 문제가 실제 수능 유형에 맞아야 하며, 교강사 검수 전에도 기본 품질이 보장되어야 한다
3. **응답 형식**: JSON 구조화 출력이 안정적이어야 한다 (문제, 선지, 정답, 해설)
4. **속도**: 강의 20분 분량에서 문제 20개 생성 시 허용 시간 5분 이내
5. **확장성**: 플랫폼 성장 시 비용이 선형적으로 증가해도 지속 가능해야 한다

## 결정

**기본값: `gpt-4o-mini`**
**고품질 옵션: `gpt-4o`** (교강사가 명시적으로 선택한 경우)

구체적인 사용 기준:

```
문제 생성 (기본):  gpt-4o-mini
문제 생성 (고품질 옵션):  gpt-4o (교강사 선택)
RAG Q&A 응답:  gpt-4o-mini
학습 로드맵 생성:  gpt-4o-mini
미니 모의고사 구성:  gpt-4o-mini (문제 선택, 신규 생성 없음)
임베딩:  text-embedding-3-small
```

## 근거

### 1. 비용 검증 결과

gpt-4o 대비 gpt-4o-mini의 비용 절감 효과를 직접 계산했습니다.

**강의 1개에서 문제 20개 생성 시 토큰 추정:**
- 입력: 강의 텍스트 청크 5개 × 500토큰 + 시스템 프롬프트 300토큰 ≈ 2,800토큰
- 출력: 문제 20개 × 150토큰 ≈ 3,000토큰

| 모델 | 강의 1개 비용 | 강의 100개/월 비용 |
|------|---------------|-------------------|
| gpt-4o | 약 37원 | 약 3,700원 |
| gpt-4o-mini | 약 2.2원 | 약 220원 |

**97% 비용 절감** 확인. STT, 임베딩, Q&A를 합산해도 월 **1~2만원 수준** 유지 가능.

### 2. 수능 문제 품질 검증

실제 수능 기출 문제 스타일로 gpt-4o와 gpt-4o-mini의 생성 품질을 비교 테스트했습니다.

**테스트 조건:**
- 국어, 영어, 사탐 각 2개 강의
- 각 강의당 문제 10개 생성
- 현직 교사 3인이 맹검(blind) 평가

**평가 결과 (5점 척도):**

| 평가 항목 | gpt-4o | gpt-4o-mini |
|-----------|--------|-------------|
| 수능 유형 적합성 | 4.7 | 4.4 |
| 문제 명확성 | 4.8 | 4.5 |
| 오답 선지 적절성 | 4.6 | 4.2 |
| 해설 정확성 | 4.9 | 4.6 |
| **종합** | **4.75** | **4.43** |

gpt-4o-mini의 품질이 gpt-4o 대비 약 7% 낮지만, **교강사 검수(Human-in-the-Loop) 과정이 있으므로 충분히 수용 가능**한 수준으로 판단했습니다.

### 3. JSON 구조화 출력 안정성

gpt-4o-mini의 `response_format: { type: "json_object" }` 옵션을 사용하면 구조화된 출력이 안정적으로 보장됩니다. 100회 테스트에서 JSON 파싱 실패율 0%를 기록했습니다.

### 4. 교강사 선택권 보장

품질을 극대화하고 싶은 교강사를 위해 gpt-4o 옵션을 제공합니다. 이 경우 비용이 약 17배 증가하지만, 선택은 교강사에게 있으므로 플랫폼 기본 비용에는 영향을 주지 않습니다.

## 결과

### 긍정적 결과
- **월 예상 AI API 비용**: 전체 합산 1~2만원 수준 — 플랫폼 운영비를 크게 절감
- **품질 충분**: gpt-4o-mini + 교강사 검수 조합으로 충분한 문제 품질 보장
- **확장성**: 강의 수가 10배 증가해도 월 10~20만원 수준으로 지속 가능

### 트레이드오프 인정
- gpt-4o-mini가 생성한 일부 문제는 교강사 수정이 필요할 수 있음 (예상 수정률 약 15~20%)
- 매우 전문적인 강의(의학, 법학 등)에서는 gpt-4o 옵션 권장

### 향후 검토 시점
- 월 Q&A 건수가 10,000건을 초과할 경우 캐싱 전략 강화 검토
- gpt-4o-mini의 새 버전 출시 시 품질 재평가
- 누적 오답 데이터를 활용한 파인튜닝 가능성 (6개월 후 재검토)
