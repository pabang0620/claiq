# CLAIQ 경쟁사 분석

> **심사 피드백 반영**: "콴다·클래스팅은 실제 경쟁군이 아니다"는 지적을 수용하여 실제 경쟁 구도를 재분류합니다. CLAIQ는 수능 학원 B2B 플랫폼이므로, 경쟁사를 직접 경쟁(수능 AI 도구)과 간접 경쟁(학원 관리 SaaS)으로 구분합니다.

---

## 실제 경쟁 서비스 분류

### 직접 경쟁: 수능 학원 AI 도구

| 서비스 | 핵심 기능 | CLAIQ 대비 차이점 |
|--------|----------|-----------------|
| 메가스터디 엘리하이 | 학생 1:1 AI 과외 | B2C 개인 대상, 학원 관리 기능 없음 |
| 이투스 AI 튜터 | 오답 분석, 유형별 피드백 | 학생 전용, 교강사 연동 없음 |
| 대성마이맥 | 강의 스트리밍 + 문제 풀기 | 직접 제작 콘텐츠 의존, AI 생성 없음 |

### 간접 경쟁: 학원 관리 SaaS

| 서비스 | 핵심 기능 | CLAIQ 대비 차이점 |
|--------|----------|-----------------|
| 클래스팅 | 출결, 과제, 소통 관리 | AI 문제 생성 없음, 수능 특화 없음 |
| 콴다 | 학생 질문 크라우드소싱 답변 | 교강사 개인 강의 연동 없음 |
| 아이엠스쿨 | 학원 알림장, 출결 | AI 기능 없음 |

## CLAIQ 차별화 포인트

### 1. 강의 녹음 → 문제 자동 생성 (Unique)

기존 서비스는 모두 사전 제작된 문제 풀을 활용합니다. CLAIQ는 각 학원 교강사의 실제 강의 내용에서 수능 형식 문제를 자동 생성합니다. **학원 고유 커리큘럼이 그대로 반영**되는 방식으로, 어떤 경쟁 서비스도 제공하지 않는 기능입니다.

### 2. 3-role 생태계 통합 (Unique)

운영자·교강사·수강생이 단일 플랫폼 안에서 연결됩니다. 강의 업로드부터 이탈 감지, 학부모 SMS까지 원스톱으로 처리됩니다. 기존 서비스는 학생 대상 또는 관리 도구 중 하나에만 집중하여 역할 간 단절이 발생합니다.

### 3. D-day 역산 우선순위 (Unique)

수능까지 남은 일수를 기반으로 약점 유형 학습 우선순위를 자동 배분합니다. 타 서비스는 약점을 식별하는 데 그치고 우선순위는 학생 판단에 맡기지만, CLAIQ는 "지금 무엇을 먼저 해야 하는가"까지 AI가 결정합니다.

### 4. AI 비용 투명성 (Differentiator)

강의 1편당 OpenAI API 비용을 명시합니다(~$0.19). 교강사가 예산 범위 내에서 AI 활용을 직접 조절할 수 있어, B2B SaaS로서의 신뢰도를 높입니다.

## 글로벌 유사 서비스와의 차별점

| 서비스 | 국가 | 핵심 기능 | CLAIQ와 차이점 |
|--------|------|---------|--------------|
| Khanmigo (Khan Academy) | 미국 | AI 튜터링, 학생 1:1 대화 | 학원 B2B 없음, 교강사 커리큘럼 연동 불가 |
| Class Companion | 미국 | AI 글쓰기 피드백, 교사 도구 | 수능 유형 매핑 없음, 한국어 특화 없음 |
| Diffit | 미국 | 교사용 학습자료 AI 생성 | 강의 녹음→문제 생성 파이프라인 없음 |
| Magic School AI | 미국 | 교사 행정업무 자동화 | 학생-교강사-운영자 생태계 통합 없음 |

### CLAIQ만의 포지션
글로벌 서비스들은 **학생 개인** 또는 **교사 개인** 도구에 집중합니다. CLAIQ는 **한국 수능이라는 고정 목표 시험** + **학원 B2B 생태계 (운영자·교강사·수강생·학부모 4역할)** 를 통합한 유일한 플랫폼입니다.

## 경쟁 포지셔닝 요약

```
                  학생 개인 대상
                        │
          이투스 AI ●  │  ● 메가스터디 엘리하이
          대성마이맥 ●  │
                        │
AI 기능 없음 ───────────┼─────────────── AI 기능 있음
                        │
          클래스팅 ●   │          ● CLAIQ
          아이엠스쿨 ● │        (목표 포지션)
                        │
                  학원/기관 대상
```

CLAIQ는 "AI 기능 × 학원/기관 대상" 사분면에서 경쟁 공백을 공략합니다. 현재 이 포지션을 점유한 직접 경쟁자가 없으며, 이것이 CLAIQ의 시장 기회입니다.
