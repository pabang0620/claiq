# CLAIQ 기능 테스트 플랜

> **테스트 환경**: https://claiq.vercel.app  
> **AI 테스트**: Playwright 자동 실행 결과 (2026-04-12)  
> 범례: ✅ 통과 | ❌ 오류 | ⚠️ 부분/제한 | — 미테스트  
> **주의**: SPA 특성상 로그인 후 `page.goto()` 직접 접근 시 인증 세션 초기화 → 로그인 리다이렉트 발생. 2차 재테스트는 사이드바 클릭 이동 방식으로 전면 재실행.
