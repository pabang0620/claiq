## 서비스 개요

CLAIQ(클레이크)는 수능 준비 중소 학원을 위한 AI 통합 교육 플랫폼입니다.  
교강사의 **강의 녹음 → AI 문제 자동 생성 → 학생 맞춤 학습 로드맵**까지 원스톱으로 제공합니다.

| 구분 | 내용 |
|------|------|
| 대상 | 수능 준비 중소 학원 (교강사 · 수강생 · 운영자) |
| 핵심 AI | GPT-4o, Whisper-1, text-embedding-3-small |
| 기술 스택 | React 19 · Vite 7 · Node.js · Express · PostgreSQL + pgvector |

## 공통

### 로그인 `/login`
- 이메일 · 비밀번호 인증
- 데모 계정 빠른 로그인 버튼 (5개 계정)
- JWT Access Token + Refresh Token 자동 갱신

### 회원가입 `/signup`
- 2단계 플로우: 역할 선택(교강사 / 수강생 / 운영자) → 정보 입력
- 운영자는 학원명 추가 입력 → 학원 자동 생성
- 비밀번호 강도 검증 · 이용약관 동의

## 교강사

### 1. 대시보드 `/teacher`
| 위젯 | 설명 |
|------|------|
| 주간 업로드 | 이번 주 강의 업로드 횟수 |
| 검증 대기 | AI 생성 문제 중 검토 필요 건수 |
| 주간 출석률 | 이번 주 학원 전체 출석률 (%) |
| 에스컬레이션 | AI가 답변 못한 학생 질문 수 |

- 빠른 액션 카드: 강의 업로드 · 문제 검증 · 에스컬레이션 · 출결 관리
- 최근 업로드 강의 목록 + 처리 상태 표시

---

### 2. 강의 업로드 `/teacher/upload` ⭐ AI
**강의 녹음 → Whisper STT → GPT-4o 문제 자동 생성**

1. 음성 파일 업로드 (드래그앤드롭 · 최소 1분 이상 검증)
2. 강의 제목 · 과목 선택
3. 실시간 처리 현황 SSE 스트리밍 표시
   - `파일 업로드 완료 → 음성 분석 중 → 수능 유형 매핑 → 문제 생성 중 → 완료`
4. 완료 후 생성된 문제 개수 표시 및 검증 페이지 링크 안내

| AI 처리 흐름 | 설명 |
|------|------|
| Whisper-1 | 음성 파일 → 텍스트 전사 (STT) |
| TypeMapper | 강의 내용 → 수능 기출 유형 코드 자동 매핑 |
| GPT-4o | 수능 유형별 문제 자동 생성 (난이도 상/중/하 × 유형별) |

> 업로드된 음성 파일은 강의 자료 페이지에도 자동 등록됩니다.

---

### 3. 문제 검증 `/teacher/review` ⭐ AI (Human-in-the-Loop)
**AI 생성 문제의 품질을 교강사가 직접 검토**

- 탭별 목록: **검증 대기 · 승인됨 · 반려됨** (각 탭 문제 수 실시간 표시)
- 2열 그리드 레이아웃으로 한눈에 비교
- 각 문제 카드 액션:
  - **승인**: 학생에게 출제 가능 상태로 전환
  - **수정 후 승인**: 문제 내용 · 선택지 · 해설 직접 수정 → 승인
  - **반려**: 출제 제외
- 문제 펼치기: 선택지 · 정답 · 해설 확인 (5지선다 정답 하이라이트)
- 강의 출처 표시 (어느 강의에서 생성된 문제인지)

---

### 4. 출결 관리 `/teacher/attendance`
- 날짜 선택 (과거 날짜만 가능, KST 기준)
- 강의 선택 드롭다운

- 기존 기록 수정 · 신규 기록 생성

---

### 5. 질문 에스컬레이션 `/teacher/escalation` ⭐ AI 연계
**AI Q&A가 답변하지 못한 질문을 교강사가 직접 답변**

- 탭별 목록: **미답변 · 답변 완료**
- 학생 질문 원문 표시 (AI 대화 컨텍스트 포함)
- 답변 입력 후 등록 → 학생 Q&A 대화에 반영

---

### 6. 강의 자료 관리 `/teacher/material`
- 강의 선택 드롭다운 (최대 50개)
- 자료 업로드: PDF · PPT · PPTX · Word · 이미지 (JPG/PNG/WEBP/GIF)
- 자료 목록: 파일명 · 크기 · 업로드일
- 자료 다운로드 (Blob 방식, 크로스오리진 대응)

## 시스템 특징

### 실시간 처리
- **SSE**: 강의 업로드 → AI 처리 진행상황 실시간 스트리밍
- **스트리밍 응답**: Q&A 챗봇 GPT 응답 실시간 출력
- **타이머**: 모의고사 20분 제한 + 자동 제출

### 보안 · 안정성
- JWT Access Token (1시간) + Refresh Token Rotation (30일, httpOnly cookie)
- Parameterized Query (SQL Injection 방지)
- idempotency key 기반 중복 포인트 지급 방지
- 역할별 라우트 보호 (teacher · student · operator)

### 시간대
- 서버 · DB · 크론 작업 모두 `Asia/Seoul` (KST) 기준
- 프론트엔드 날짜 계산 `Intl.DateTimeFormat('en-CA', { timeZone: 'Asia/Seoul' })`

### 게임화 (Gamification)
- 포인트 시스템: 출석 · 문제풀이 · 스트릭 · 목표 달성 시 자동 지급
- 뱃지 컬렉션: 7가지 뱃지 + 완성 보상 500P
